# Asaka Campus Portal

> **⚠️ このサイトは現在テスト中です。まだ正式にリリースされていません。**

東洋大学朝霞キャンパスの学生向けポータルサイトです。学生生活に必要な情報をまとめています。

## 🔍 情報の探し方

### 🔎 まずは検索を使ってみましょう

このサイトは **全文検索機能** を搭載しています。ページ上部の検索ボックスに知りたいキーワードを入力するだけで、関連する情報を素早く見つけることができます。

**検索のコツ：**
- 「奨学金」「履修登録」「証明書」など、知りたい内容のキーワードを入力
- 複数のキーワードを組み合わせて絞り込みも可能
- 手続き方法や施設情報など、あらゆるページから検索できます

### 📋 メニューから探す

左側のメニューから、カテゴリごとに情報を閲覧することもできます。各ページには詳細な手続き方法や利用案内が記載されています。
