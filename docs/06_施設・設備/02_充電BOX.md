# 充電BOX

スマートフォンやタブレットを安全に充電できる設備です。

## 📍 設置場所

- 学生ホール
- 図書館
- 食堂
- 各号館ラウンジ

## 📝 利用方法

1. 空いているロッカーを選ぶ
2. 自分の充電ケーブルを接続
3. 扉を閉めて施錠
4. 充電完了後、開錠して端末を取り出す

## 🕐 利用時間

各施設の開館時間に準じます

## 💰 利用料金

無料

## ⚠️ 注意事項

- **充電ケーブルは各自で用意してください**（貸出はありません）
- 長時間の放置は避けてください（最長3時間まで）
- 利用後は必ず端末を取り出してください
- 盗難・紛失については大学は責任を負いません
- 故障や不具合を発見した場合は施設管理課に連絡してください

## 📱 対応機器

- スマートフォン（iPhone、Android等）
- タブレット
- モバイルバッテリー
- その他USB充電可能な機器

各BOXにはUSB Type-A、Type-C、Lightningなどの主要な接続端子があります。
