# 朝霞トレーニングルーム

体力づくりや健康維持のためのトレーニング施設です。

## ✅ 利用資格

- 本学の学生・教職員
- 初回利用時にオリエンテーション受講が必要

## 🏋️ 設備

- ランニングマシン
- エアロバイク
- 筋力トレーニングマシン
- フリーウェイト
- ストレッチエリア

## 🕐 利用時間

平日：12:00 - 20:00
土曜：10:00 - 17:00
日曜・祝日：休館

**※授業期間外や長期休暇中は時間が変更される場合があります**

## 📝 利用方法

1. 学生証を持参
2. 受付で利用手続き
3. トレーニングウェア・室内シューズに着替え
4. 利用

## 💰 利用料金

無料

## ⚠️ 注意事項

- 初回利用時は必ずオリエンテーション（約30分）を受講してください
- トレーニングウェアと室内シューズが必要です
- 貴重品は各自で管理してください
- 器具の使い方がわからない場合はスタッフに確認してください
- 体調不良時の利用は控えてください

## 📅 オリエンテーション予約

保健体育センターまたはトレーニングルーム受付で申し込み
