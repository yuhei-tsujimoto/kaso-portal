# 学研災・学研賠加入証明書発行

学外実習や就職活動などで必要となる保険加入証明書の発行手続きです。

## 📚 学研災・学研賠とは

- **学研災**：学生教育研究災害傷害保険（学生自身のケガを補償）
- **学研賠**：学研災付帯賠償責任保険（他人への損害賠償を補償）

## 📋 加入証明書が必要な場面

- 学外実習参加時
- インターンシップ参加時
- 教育実習参加時
- ボランティア活動参加時
- 就職活動時

## 📝 発行方法

1. 学務課窓口で申請書を記入
2. 学生証を提示
3. 即日発行（通常5分程度）

## 💰 費用

無料（保険料は入学時に一括納入済み）

## ⚠️ 注意事項

- 全学生が自動加入しています
- 保険期間は在学期間中
- 証明書の有効期限を確認してください
- 紛失した場合は再発行可能（無料）
