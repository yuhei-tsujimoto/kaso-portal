# 休学・退学・復学申請

学籍に関する重要な手続きについての情報です。  

## 休学

病気、留学、経済的理由などで一時的に修学を中断する制度です。  

### ⏱️ 休学期間

- 1学期単位または1年単位
- 最長2年間まで（通算）
- 休学期間は在学期間に算入されません

### 📝 申請方法

1. 学務課で相談
2. 指導教員と面談
3. 休学願を提出
4. 学費の取り扱いについて確認

### ⏰ 申請期限

- 前期休学：3月31日まで
- 後期休学：9月30日まで

## 復学

休学期間満了後、または休学期間中に復学する手続きです。  

### 📝 申請方法

1. 復学予定日の1ヶ月前までに学務課に連絡
2. 復学願を提出
3. 履修指導を受ける

## 退学

やむを得ない事由により学業を継続できない場合の手続きです。  

### 📝 申請方法

1. 保証人と相談
2. 指導教員と面談
3. 学務課で退学願を受け取り
4. 必要事項を記入し提出
5. 学生証を返却

### ⚠️ 注意事項

- 退学日によって学費の取り扱いが異なります
- 奨学金を受給している場合は返還手続きが必要です
- 再入学制度について確認してください

## 相談窓口

重要な決断です。まずは学務課または学生相談室にご相談ください。
