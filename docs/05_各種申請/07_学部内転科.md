# 【生命科学部】学部内転科制度

生命科学部内で学科を変更できる制度です。

## 📋 制度概要

入学後、学びを深める中で興味・関心が変化した場合、学部内の他学科への転科が可能です。

## 🎯 対象学科

生命科学部内の各学科間

- 生命科学科
- 応用生物科学科
- 環境応用化学科
- など

## ✅ 出願資格

- 生命科学部に在籍する1年次修了者
- 所定の単位を取得していること
- 成績基準を満たしていること（GPA等）

## 📊 選考方法

- 書類審査
- 面接
- 成績審査

## 🔄 申請の流れ

1. 指導教員に相談
2. 転科先学科の説明会に参加（開催される場合）
3. 申請書類を準備
4. 学務課に提出
5. 選考
6. 結果通知

## 📅 申請時期

毎年1月〜2月頃（詳細は掲示で確認）

## ⚠️ 注意事項

- 転科後は転科先学科のカリキュラムに従います
- 既修得単位の認定については個別に審査されます
- 定員に制限がある場合があります
- 転科により卒業が遅れる可能性があります

## 📞 相談窓口

まずは指導教員または学務課にご相談ください。
