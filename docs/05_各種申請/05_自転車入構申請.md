# 朝霞自転車入構申請

朝霞キャンパスに自転車で通学する際の登録手続きです。

## 💡 申請が必要な理由

キャンパス内の安全管理と駐輪場の適切な利用のため、自転車通学には事前登録が必要です。

## 📝 申請方法

1. 学務課で申請書を受け取る
2. 必要事項を記入
3. 自転車の防犯登録番号を記入
4. 学務課に提出
5. 駐輪許可証（ステッカー）を受け取る

## 📋 必要なもの

- 学生証
- 自転車の防犯登録番号がわかるもの

## 💰 費用

無料

## 🅿️ 駐輪場所

指定された駐輪場のみ利用可能です。許可証を自転車に貼付してください。

## ⚠️ 注意事項

- 許可証のない自転車は撤去される場合があります
- 駐輪場以外への駐輪は禁止です
- 盗難防止のため必ず施錠してください
- 許可証を紛失した場合は再発行が必要です（無料）
- 自転車を変更した場合は再度申請が必要です

## 🔄 更新

毎年度更新が必要です（4月に新しい許可証を配布）
