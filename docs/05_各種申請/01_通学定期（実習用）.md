# 実習用通学定期発行申請

学外実習に参加する際の通学定期券購入に必要な証明書の発行手続きです。  

## 対象者

- 学外実習に参加する学生
- 実習先への交通費が発生する学生

## 申請方法

1. 学務課で申請書を受け取る
2. 実習先の情報を記入
3. 指導教員の承認を得る
4. 学務課に提出

## 必要書類

- 実習用通学定期発行申請書
- 実習スケジュール表

## 申請期限

実習開始の2週間前まで  

## 注意事項

- 通常の通学定期とは異なる手続きです
- 実習期間に応じた定期券が購入可能
- 証明書の発行には3〜5営業日かかります
