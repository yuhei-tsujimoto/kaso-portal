# 研究生制度

大学卒業後に特定のテーマについて研究を継続するための制度です。

## 📚 研究生とは

正規の学生ではありませんが、大学の研究施設を利用して研究活動を行うことができる制度です。学位の取得はできません。

## ✅ 出願資格

- 大学を卒業した者、または同等以上の学力があると認められた者
- 本学で研究を希望する特定のテーマがあること
- 指導を受け入れる教員がいること

## ⏱️ 研究期間

- 1年間（更新可能、最長2年まで）
- 年度途中からの入学も可能

## 📝 出願方法

1. 希望する研究分野の教員に連絡・相談
2. 受け入れ承諾を得る
3. 出願書類を準備
4. 学務課に提出

## 📄 必要書類

- 研究生入学願
- 履歴書
- 最終学歴の卒業証明書
- 成績証明書
- 研究計画書
- 指導教員の承諾書

## 💰 検定料・授業料

- 検定料：約10,000円
- 授業料：学期ごとに約150,000円（金額は年度により異なる）

## ⚠️ 注意事項

- 学位は授与されません
- 研究生の在籍期間は正規課程への出願資格には含まれません
- 大学院進学を希望する場合は、別途入学試験を受験する必要があります
- 施設・設備の利用については指導教員の指示に従ってください

## 📞 相談窓口

希望する研究分野の教員または学務課までお問い合わせください。
