# 臨床工学技士

医療現場で生命維持管理装置の操作・保守を担う国家資格です。

## 📋 資格概要

臨床工学技士は、人工心肺装置、人工透析装置、人工呼吸器などの生命維持管理装置の操作および保守点検を行う医療専門職です。

## ✅ 受験資格

本学の指定科目を履修し、所定の単位を取得することで受験資格が得られます。

## 🏢 主な就職先

- 総合病院
- 大学病院
- クリニック
- 医療機器メーカー
- 医療機器販売会社

## 📅 試験時期

毎年3月（年1回実施）

## 🤝 サポート体制

- 国家試験対策講座
- 模擬試験
- 個別指導

詳細は学務課またはキャリアセンターにお問い合わせください。
