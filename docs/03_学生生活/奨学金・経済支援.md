# 奨学金・経済支援

## 奨学金制度

### 📝 日本学生支援機構（JASSO）奨学金

- **申請時期**: 4月（定期採用）、随時（緊急・応急採用）
- **種類**:
    - 給付奨学金（返還不要）
    - 貸与奨学金（第一種・第二種）
- **窓口**: 学生部学生課

### 🏫 東洋大学独自奨学金

- **種類**:
    - 成績優秀者奨学金
    - 経済的支援奨学金
    - 課外活動奨学金
- **申請**: ToyoNet-Gで案内

## その他の経済支援

### 💳 授業料減免制度

- 経済的理由により修学が困難な学生を対象
- 前期・後期の年2回申請可能

### 🏢 学生寮・住居支援

- 提携学生寮の案内
- 住居費補助制度

## 相談窓口

- **場所**: 朝霞キャンパス 学生課
- **対応時間**: 平日 9:00-17:00
