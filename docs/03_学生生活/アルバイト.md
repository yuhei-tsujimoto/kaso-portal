# アルバイト

## 学内アルバイト

### 📚 図書館スタッフ

- カウンター業務、書架整理など
- 募集時期: 3月・9月
- ToyoNet-Gで募集案内

### 🖥️ TA（ティーチング・アシスタント）

- 授業補助、実験・実習のサポート
- 大学院生が主な対象

### 🏢 事務補助

- 各部署での事務作業サポート
- 募集は不定期

## 学外アルバイト紹介

### 📱 アルバイト情報サイト

- ToyoNet-G「アルバイト情報」
- 大学が審査した求人情報を掲載
- 安全性・学業との両立を考慮

### ⚠️ 注意事項

- 悪質な勧誘に注意
- 労働条件を必ず確認
- トラブル時は学生課に相談

## アルバイトと学業の両立

### ⏰ 推奨勤務時間

- 週20時間以内が目安
- 試験期間は控えめに

### 📖 学業優先の原則

- 授業・試験を最優先
- シフト調整が可能な職場を選ぶ

## 禁止事項・注意事項

### ❌ 避けるべきアルバイト

- 危険を伴うもの
- 法律・公序良俗に反するもの
- 深夜勤務（22時以降）の頻度が高いもの

### 📝 労働条件の確認

- 雇用契約書の確認
- 最低賃金の遵守
- 労働時間・休憩時間

## 相談窓口

- **学生課**: アルバイト全般の相談
- **学生相談室**: 労働トラブル、両立の悩み
