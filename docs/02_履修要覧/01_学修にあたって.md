# 学修にあたって

## 学修の基本方針

大学での学修は、主体的な学習姿勢が重要です。授業への積極的な参加と、授業時間外の予習・復習を通じて、深い理解を目指しましょう。

## 学習環境の整備

### 📖 シラバスの確認

各科目のシラバスを事前に確認し、授業の目標、内容、評価方法を理解しておくことが重要です。

### 💻 学習ツールの活用

- ToyoNet-G：履修登録、成績確認、シラバス閲覧
- ToyoNet-ACE：授業の教材配信、レポート提出
- 各学科Webサイト：教育課程表、卒業要件の確認

## 学習サポート

### 🧑‍💼 ラーニングサポートセンター

授業で理解できなかった点や疑問点について気軽に質問できる場です。

- **場所**: 朝霞キャンパス1号館 2階
- **開室日時**: 毎週 月～金曜日 8:00～21:30

### 🕰️ オフィスアワー

教員と直接コミュニケーションを取るための時間帯です。授業内容に関する質問や学習方法の相談などに活用できます。
