# 成績評価

## 成績の評価方法

### 📈 評価基準

成績は、S、A、B、C、D、Eの6段階および評価不能（＊）で表示されます。

- **S**: 90点以上（優秀）
- **A**: 80点以上（良好）
- **B**: 70点以上（可）
- **C**: 60点以上（可）
- **D**: 50点以上（不可）
- **E**: 50点未満（不可）
- **＊**: 評価不能

### 🎯 GPA制度

GPA（Grade Point Average）は、各科目の単位数を考慮した重み付き平均点です。成績表に表示されます。

## 成績の確認方法

### 📋 成績照会

ToyoNet-G「学生メニュー」→「成績照会」から確認できます。

### 📅 成績発表日

各学期終了後、成績発表日が設定されます。詳細は朝霞キャンパスポータルサイトのスケジュールで確認してください。

## 成績調査申請

### 📝 申請方法

成績評価基準を満たしているのに評価に疑問がある場合、成績発表日直後に設定される申請期間内に、ToyoNet-G「学生メニュー」→「成績照会」→「成績調査申請」から申請を行えます。

### ⚠️ 注意事項

成績調査は成績の再考や個別の得点開示を求めるものではありません。本学の成績評価基準および各科目のシラバスに記載された「成績評価の方法・基準」を必ず確認したうえで申請してください。

### 📅 結果の確認

調査結果は成績調査結果発表日に同じメニューから確認できます。

## 単位認定

### ✅ 単位修得

成績がC以上の場合、単位が認定されます。

### 🔄 再履修

単位未修得の場合、再度履修（同一科目の再登録）が可能です。
