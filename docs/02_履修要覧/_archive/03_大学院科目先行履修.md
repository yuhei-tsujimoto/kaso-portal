# 大学院開講科目履修制度（先行履修）

学部4年次に大学院の授業を履修できる制度です。取得した単位は、大学院進学後に認定されます。  

## 対象者

- 学部4年生
- 本学大学院への進学を予定している学生
- 成績優秀者（GPA基準あり）

## 履修可能科目

大学院で開講されている科目のうち、指導教員が認めたもの  

## 申請方法

1. 指導教員に相談
2. 大学院担当教員の承認を得る
3. 所定の申請書を学務課に提出

## 申請期限

各学期の履修登録期間内  

## メリット

- 大学院進学後の研究に早期着手できる
- 進学後の履修負担を軽減できる
- 専門性の高い知識を早期に習得できる
