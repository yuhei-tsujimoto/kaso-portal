# 試験

## 試験の種類

### 📋 定期試験

各学期末に実施される試験です。試験期間は授業期間終了後に設けられています。

### 📝 授業内試験

授業時間内に実施される試験です。シラバスで確認してください。

### 📄 レポート試験

レポート提出による評価が行われる場合があります。提出期限や形式は各科目のシラバスで確認してください。

## 試験の確認方法

### 📅 試験スケジュール

試験の日程や教室は、ToyoNet-G「学生メニュー」→「履修・授業照会」→「授業時間割照会」で確認できます。

### 📢 試験に関するお知らせ

東洋大学公式アプリ及びToyoNet-Gで通知されます。

## 試験の注意事項

### ⏰ 試験開始時刻の厳守

試験開始時刻に遅刻した場合、受験できない場合があります。余裕を持って会場に到着してください。

### 📚 持ち込み可能なもの

各科目によって異なります。シラバスや試験前の指示を確認してください。

### 🚫 不正行為の禁止

カンニングや不正行為は厳禁です。発覚した場合、単位不認定や処分の対象となります。

## 再試験・追試験

### 🔄 再試験

単位未修得の場合、再試験が実施される場合があります。詳細は各科目の担当教員に確認してください。

### 📅 追試験

やむを得ない事情で定期試験を受験できなかった場合、追試験が認められる場合があります。事前に担当教員に相談してください。
