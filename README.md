# Asaka Campus Portal

東洋大学朝霞キャンパスポータル - 学生生活に必要な情報をまとめたドキュメントサイトです。

## 🌐 サイト

**https://yuhei-tsujimoto.github.io/asaka-portal/**

## 📚 コンテンツ

- **スケジュール** - 年間行事予定や授業日程など
- **履修・成績** - 履修登録や成績に関する情報
- **学生生活** - 奨学金、サークル、施設利用など
- **進路・キャリア** - 就職支援やインターンシップに関する情報
- **各種申請** - 証明書発行や届出用紙のダウンロード

## 🛠️ 技術スタック

- [Zensical](https://zensical.org/) - 静的サイトジェネレーター
- GitHub Pages - ホスティング
- GitHub Actions - CI/CD
